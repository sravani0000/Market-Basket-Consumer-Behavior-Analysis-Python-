Project Overview: Optimized retail cross-selling strategies by identifying frequent item sets and association rules within large transactional logs.

Technical Highlights:

Association Rule Mining: Implemented the Apriori Algorithm using the mlxtend library to discover hidden patterns in consumer behavior.

KPI Evaluation: Filtered rules based on Lift (>1.0) and Confidence to ensure only statistically significant product pairings were recommended for marketing interventions.

ETL Automation: Automated the transformation of raw log data into structured dataframes suitable for recurring monthly business reviews.
